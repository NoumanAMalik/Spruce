# Spruce

A minimal issue tracker with native AI that transforms rough ideas into detailed, actionable tickets.

## Overview

Spruce is an issue and feature tracker designed for teams that need to capture ideas quickly without losing detail.

Users submit ideas in natural language through a chat-style interface, optionally with screenshots. The AI analyzes the project context and enhances the submission with relevant technical details, acceptance criteria, and connections to existing issues.

## Core Features

- **Chat-based ticket creation** — Submit issues and feature requests through a conversational interface
- **AI-powered enhancement** — Automatically expands brief inputs into comprehensive tickets with project-aware context
- **Minimal interface** — Clean layout with zero learning curve
- **Screenshot support** — Attach images directly when creating tickets

## Design Principles

- Low friction adoption for existing team workflows
- No templates or required fields
- Fast capture, detailed output
